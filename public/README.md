# Public Assets Folder

Place your logo image file here.

## Logo File

Name your logo file: `logo.png`

The logo should be a complete image containing:
- The circular emblem with "KENYA METHODIST UNIVERSITY" text
- The "KeMU" text
- The "Kenya Methodist University" text

Supported formats: `.png`, `.jpg`, `.jpeg`, `.svg`

The logo will automatically scale to fit the navbar height (64px on mobile, 80px on desktop) while maintaining its aspect ratio.
